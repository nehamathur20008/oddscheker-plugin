# WP Odds Comparison Plugin

The WP Odds Comparison Plugin allows you to compare live odds from multiple bookmakers directly on your WordPress site. The plugin fetches data from popular odds comparison platforms and displays it in an easy-to-understand format.

## Features

- **Live Odds Fetching**: The plugin retrieves live odds from multiple bookmakers like OddsChecker, Betfair, and others.
- **Admin Dashboard**: A customizable interface to select which bookmakers to display, manage market types, and update bookmaker links.
- **Gutenberg Block**: A Gutenberg block that can be added to WordPress posts and pages, allowing users to dynamically add odds comparisons with selected bookmakers.
- **Odds Conversion**: The plugin includes an odds converter, which allows users to convert odds between fractional, decimal, and American formats.

## Installation

1. **Download the Plugin**: Download the plugin as a `.zip` file.
2. **Upload the Plugin**:
   - Go to your WordPress Admin Dashboard.
   - Navigate to **Plugins > Add New**.
   - Click **Upload Plugin** and select the `.zip` file you downloaded.
   - Click **Install Now** and then **Activate**.
   
3. **Configure the Plugin**:
   - After activation, go to the **Odds Comparison Settings** in the admin menu.
   - Select the bookmakers and markets you want to display.
   - Save the settings.

## Usage

- **Adding to Posts/Pages**: You can add the odds comparison block to your posts or pages using the Gutenberg editor. Simply add the "Odds Comparison" block, select the bookmakers you wish to compare, and publish your post.
  
- **Viewing Odds**: On the front-end of your website, the plugin will display live odds in a dynamic comparison format.

## Settings

The settings for the plugin can be found in the **Odds Comparison Settings** section of the admin dashboard. You can:
- Select which bookmakers to display odds from.
- Choose the specific markets to display.
- Configure the links to the bookmakers for easy navigation.
